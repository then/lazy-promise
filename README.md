# lazy-promise

  A bare-bones lazily-evaluated Promises/A+ implementation.

## Installation

    $ npm install lazy-promise

  or

    $ component install then/lazy-promise

## API

### new Promise(function(resolve, reject))

  Create a new lazy promise with the given function.
